# Project Brief: Building Agency

## Описание проекта
Веб-сайт жилого комплекса «Строительный комплекс» — интерактивная схема этажей для просмотра и бронирования квартир в режиме реального времени.

## Цель
Единая страница с интерактивной схемой всего жилого комплекса (12 секций, 15 этажей) для выбора и бронирования квартир.

## Источник данных
Реальные чертежи здания (22 плана) из папки `/plan-zdaniya/schemaImages/`:
- Планы 1-го этажа и типовых этажей (2-15) для всех 12 блоков
- Точные площади квартир, планировки, экспликация помещений
- Железобетонный каркас, сетка колонн, лестнично-лифтовые узлы

Файлы чертежей скопированы в `public/schemaImages/` для отдачи Next.js.

## Технический стек
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Icons**: Lucide React
- **Utils**: clsx + tailwind-merge

## Структура проекта
- `/app` - страницы Next.js
- `/components` - React компоненты
- `/data` - данные квартир (сгенерированные из чертежей)
- `/types` - TypeScript интерфейсы
- `/lib` - утилиты
- `/plan-zdaniya` - исходные чертежи и контекст
- `/public/schemaImages` - чертежи для отдачи клиенту (22 PNG файла)

## Конфигурация здания
- **Этажи**: 15 (1 + типовые 2-15)
- **Секции**: 12 (пронумерованы 1-12)
- **Типы секций**:
  - Узкие (5 квартир/этаж): Секции 1, 6, 9, 10
  - Широкие (6 квартир/этаж): Секции 2, 3, 4, 5, 7, 8, 11, 12
- **Всего квартир**: ~168 (расчётное)
- **Цена за м²**: 130 000 ₽

## Статусы квартир (только 2)
- **free** (зелёный) — доступна для бронирования
- **sold** (красный) — занята

## Классы квартир (по планировке)
- А — 2-комнатная, ~81.33 м²
- Б — 2-комнатная, ~74.45 м²
- В — 2-комнатная, ~75.57 м²
- Г — 1-комнатная, ~45.93 м²
