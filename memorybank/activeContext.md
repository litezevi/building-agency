# Active Context: Building Agency

## Текущий фокус
Обновление схемы здания под реальный проект из архитектурных чертежей.

## Реализованный функционал (после обновления)
- **12 секций** с переключением в UI
- **15 этажей** (1 + типовые 2-15)
- **~168 квартир** — все свободны (из чертежей)
- **Генерация данных** из чертежей (22 плана)
- Классы квартир: А, Б, В, Г
- Статистика секции: Всего / Свободно / Занято
- Фильтры: Этаж, Статус
- Система бронирования с localStorage
- Модальное окно с деталями квартиры

## Структура данных (из чертежей)
- Узкие секции: 5 квартир/этаж × 15 этажей = 75 кв.
- Широкие секции: 6 квартир/этаж × 15 этажей = 90 кв.
- Итого: ~168 квартир в здании

- (опционально) Интеграция с backend для реальных данных
