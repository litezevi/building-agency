# Active Context: Building Agency

## Текущий фокус
Проверка и исправление схемы здания под реальные архитектурные чертежи. Обнаружены и исправлены несоответствия в маппинге файлов и конфигурации секций.

## Реализованный функционал
- **12 секций** с переключением в UI
- **15 этажей** (1 + типовые 2-15)
- **216 квартир** — все свободны (из чертежей)
- **Генерация данных** из чертежей (22 плана)
- **Архитектурные чертежи** — показаны для каждой секции
  - План 1 этажа (thumbnail + preview при клике)
  - План типового этажа (2-15) (thumbnail + preview при клике)
- Классы квартир: А, Б, В, Г
- Статистика секции: Всего / Свободно / Занято
- Фильтры: Этаж, Статус
- Система бронирования с localStorage
- Модальное окно с деталями квартиры

## Структура данных (из context.md)
- Узкие секции (5 кв): 4 секции × 75 кв = 300 кв.
- Широкие секции (6 кв): 6 секций × 90 кв = 540 кв.
- Блок 4/5 (8 кв): 2 секции × 120 кв = 240 кв.
- **Итого: 216 квартир** (исправлено из ~168)
